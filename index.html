Hexo 简介

Hexo是一款基于Node.js的快速、简洁且高效的博客框架，Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。该文章是基于win10搭建的描述。

安装

Hexo 相当简单。首先需要安装Node.js，然后再进行Hexo安装，如果需要链接远程仓库比如外挂在Github上还需要本地安装git。

  • 安装Node.js(用来生成静态页面)

    1. 下载Node.js: 打开官网下载链接: Node.js
    2. 下载完成后，双击下载的.msi文件，开始安装Node.js，中间可以修改一下安装路径一路next就可以了
    3. Node.js已经安装完成，可以先进行下简单的测试安装是否成功了，按下【win+R】键，输入cmd，输入node -v会看到对应的版本号即安装完成
    4. Node.js更多 菜鸟教程
