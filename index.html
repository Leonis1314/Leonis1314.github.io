Git Command

Git Command

  1. 遇到错误–SSL certificate problem: unable to get local issuer certificate

    解决办法： git config –global http.sslVerify false

    描述：这样设置之后就可以重新输入用户名和密码了

  1. git不是内部或外部命令

    解决办法：windows中环境变量–系统变量–path添加：Git\mingw64\libexec\git-core和Git\bin路径

  1. 项目切换远程仓库地址命令：

    查看当前远程：git remote -v

    删除当前远程：git remote rm origin

    添加新的远程：git remote add origin url

    查看当前分支：git branch -a

    如果切换了branch远程地址，有必要的话记得切换master的，命令跟上面一样

  1. 常用命令：

    初始化：git init

    添加本地到远程：git remote add origin http://xxxxxxx

    刷新远程分支：git remote update origin –prune

    当前本地分支绑定远程分支：git branch –set-upstream-to=remotes/url

    git config –global http.sslVerify false

  1. 新建远程仓库，push本地代码时出现push to origin/master war rejected的问题解决方案：

    git pull

    git pull origin master

    git pull origin master –allow-unrelated-histories

    然后执行push即可

Logic-G610

随着无纸化办公已经成为当今社会办公的主力手段，加之各种游戏层出不穷，这就催生了大量的电脑外设井喷式的出现，各种大厂小厂纷纷推出各种旗舰产品在这一行业进行厮杀。在使用了两年后，决定记录一下这款罗技G610。

Logictech 

罗技(Logictech)这个品牌。罗技成立于1981 年，是一家起源于瑞士的全球著名云周边设备供应商。罗技公司旗下拥有罗技、罗技G、Jaybird及Ultimate Ears多个品牌，业务涉及生产力及创造力、电竞游戏、视频协作、音乐和智能家居多个领域。罗技凭借其产品出色的工业设计及创新，频频揽获各类世界级奖项。罗技的电脑外设基本都透露着低调、科技感十足的体验。作为罗技旗下的G610机械键盘自然也不例外。

G610 

罗技G610专业游戏机械键盘采用了cherry原厂MX机械轴作为机械键盘的核心组件，Cherry MX轴作为品质机械轴体的代名词，又细分为黑轴、茶轴、红轴、青轴、白轴，这个在键帽上通过相应的颜色即可区分。黑红茶青轴各有特色，如何选择就要看个人喜好了。众所周知，青、黑轴段落感最强、Click声音最大，机械感最强，是机械键盘的代表轴，需下压2.4mm才可触发，打字节奏感十 内部结构 内部结构 足，但是声音较大，比较吵 , 压力克数为60g。茶轴相比青轴则会软一些，没有那么强的段落感和Click声音，红轴则最为接近薄膜键盘手感，但也保留了机械键盘的特质。

作为一名多年的程序员，我在综合各方面之后选择了较为平和的茶轴作为日常撸码的工具。这款茶轴G610使用起来相比青轴确实柔和了不少，不过也没有丢失机械键盘应有的敲击段落感和清脆的Click声音。G610采用了光面喷漆，右上角加入了多媒体功能控制(确实比较实用)，按键加入了背光但是只是白色的单色背光，这点不足可以通过某宝购买灯罩实现炫彩的灯光效果来弥补，毕竟G610在软件控制端加入了灯光效果的配置选项。

作为日常使用的撸码工具，使用了长达两年之后依然能感受到明显的段落感，只是在Esc键出现了掉色的情况。对于一款机械键盘，罗技G610的配件有些偏少，没有拔键器、没有额余的键帽只有薄薄的三张质保卡、质保说明、说明书，不过好在购买的时候罗技官方赠送了唯一一个配件：手托。

G610另外一个优势是1.8m长的USB接口无论是办公还是游戏，都能满足。而编织网式线缆在保证线缆牢固的同时确保线缆的柔软度。

键盘背面四角+下部中央布局了5个防滑脚垫，防滑效果不错。上部脚垫内侧的2个支撑脚采用了分段设计，用户可根据按键适度需要调节键盘高度。

总之，罗技G610 Cherry MX原厂机械轴键盘保持着罗技一贯的细腻做工、精致的人体工程学设计语言，丰富的软件功能，极具性价比的机械办公伴侣，你值得拥有！

Hexo 简介

Hexo是一款基于Node.js的快速、简洁且高效的博客框架，Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。该文章是基于win10搭建的描述。
